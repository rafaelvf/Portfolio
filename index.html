Delincuenmapa

Delincuenmapa is a web application where the user is able to see which areas in a city are the most dangerous. And if the user suffered an act of crime, he can report it on the website as well. It was born out of necesity, and I think it can make a huge and positive impact in the cities around the world. The application was made in React framework, which I feel very confortable working around.I used node.js and express for the back-end. For the map I used Leaflet which I recommend a lot. It has a lot of capabilities for different types of projects. And for the database I explored MongoDb which I liked a lot. I am proud of this project because it is a mix of the reasosn why I started to code.

Clanfest

ClanFest is a web application that allows users to find, create and organize all kinds of events. Among the functionalities developed, users will be able to create and edit events, invite friends and manage event attendees, assigning them tasks or deleting them. Users will be able to find events near their location, confirm attendance or purchase tickets for the event through the integrated payment platform, Mercado Pago. ClanFest includes a notification system, part of it, integrated in the app, built with Socket.io, that will send online and offline alerts to users every time another user has interacted with their created events or events to attend, also users will receive emails to inform them of these interactions.
